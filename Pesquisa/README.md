# Pesquisa de Satisfação

Projeto de tarefa da agenda 8 da disciplina de desenvolvimento de sistemas, desenvolvido em linguagem Python para analisar a opinião de clientes sobre o atendimento de uma empresa.

## Objetivo
Realizar uma pesquisa com 50 entrevistados para avaliar o nível de satisfação no atendimento.

## Funcionamento
O programa solicita ao usuário:
- Nome
- Idade
- Avaliação do atendimento:
  - 1: EXCELENTE
  - 2: BOM
  - 3: RUIM

O sistema valida as entradas utilizando estruturas de repetição (`while`) para garantir que os dados sejam informados corretamente.

## Estruturas utilizadas
- `for`: para repetir a pesquisa 50 vezes
- `while`: para validação das entradas
- `if`: para tomada de decisão

## Resultado
Ao final da execução, o programa exibe:
- Quantidade de respostas EXCELENTE (`qtd_excelente`)
- Quantidade de respostas RUIM (`qtd_ruim`)

<-> bash
python Luis_paulo_Ag8_DS_I.py
